# 📸 사진데이터관리 웹앱 기획서

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | 사진 EXIF/GPS 데이터 관리 웹앱 |
| **목적** | 현장 사진의 촬영 위치·시간 정보를 자동 추출하여 정리·시각화 |
| **기술 스택** | Python + Streamlit |
| **배포** | GitHub → Streamlit Cloud (무료) |
| **사용 대상** | 본인 + 지인 (링크 공유) |

---

## 2. 핵심 기능

### 2-1. 사진 업로드
- 여러 장 동시 업로드 (drag & drop / 파일 선택)
- 지원 형식: JPG, JPEG, HEIC, PNG, TIFF
- EXIF 데이터가 없는 사진은 경고 표시

### 2-2. EXIF 데이터 추출 및 표시
- **촬영 일시**: 날짜 + 시각
- **카메라 정보**: 제조사, 모델명
- **촬영 설정**: 조리개, ISO, 셔터스피드
- **GPS 정보**: 위도, 경도, 고도, 촬영 방향
- **Google Maps 링크**: 클릭하면 해당 위치를 구글 지도에서 확인

### 2-3. 결과 테이블
- 업로드한 사진 목록을 표 형태로 정리
- 컬럼: 파일명 | 촬영일시 | 위도 | 경도 | 고도 | 구글지도 링크 | 카메라 모델

### 2-4. Excel 다운로드
- 추출된 데이터를 `.xlsx` 파일로 다운로드
- 파일명 예시: `사진데이터_2026-03-24.xlsx`

### 2-5. 지도 위치 표시
- 업로드한 사진들의 GPS 좌표를 지도 위에 마커로 표시
- 마커 클릭 시 파일명, 촬영일시 팝업

### 2-6. 평면도 위치 표시 (향후 확장)
- 건물/현장 평면도 이미지 업로드
- 평면도 위에 사진 촬영 위치를 수동 배치 또는 좌표 매핑
- *※ 1차 버전 이후 추가 개발 검토*

## 4. 기술 상세

### 4-1. 사용 라이브러리
| 라이브러리 | 용도 |
|-----------|------|
| `streamlit` | 웹앱 프레임워크 |
| `Pillow` (PIL) | 이미지 EXIF 읽기 |
| `pillow-heif` | HEIC 형식 지원 (iPhone) |
| `pandas` | 데이터 테이블 처리 |
| `openpyxl` | Excel 파일 생성 |
| `folium` | 지도 시각화 |
| `streamlit-folium` | Streamlit에서 folium 지도 표시 |

### 4-2. GPS 좌표 변환
- EXIF GPS 데이터: DMS 형식 (도/분/초)
- 표시 및 활용: DD 형식 (십진수) 으로 변환
- 예시: `37° 27' 13.2" N` → `37.453667`

### 4-3. Google Maps 링크 생성
```
https://www.google.com/maps?q={위도},{경도}
```

## 5. 사진 업로드 방법 및 EXIF 보존 여부

### ✅ 웹앱에 직접 업로드하면 EXIF가 보존됩니다
Streamlit 웹앱은 원본 파일을 그대로 받기 때문에 EXIF/GPS 데이터가 유지됩니다.
따라서 **USB 복사 없이도 폰/PC에서 웹앱에 바로 업로드하면 됩니다.**

### 사용 방법별 EXIF 보존 여부
| 방법 | EXIF 보존 | 비고 |
|------|----------|------|
| **웹앱에 직접 업로드 (폰/PC)** | ✅ 보존 | **권장 - 가장 간편** |
| USB 케이블로 iPhone → PC 복사 | ✅ 보존 | PC에 사진 백업 시 유용 |
| AirDrop (Mac) | ✅ 보존 | Mac 사용자 |
| iCloud.com 웹 다운로드 | ❌ 제거됨 | 사용 불가 |
| Claude 업로드 | ❌ 제거됨 | 사용 불가 |
| 카카오톡/메신저 전송 | ❌ 제거됨 | 사용 불가 |

## 6. 배포 계획

### 6-1. GitHub 저장소 구성
```
photo-data-manager/
├── app.py              # Streamlit 메인 앱
├── requirements.txt    # 의존 라이브러리
├── README.md           # 사용법 안내
└── .streamlit/
    └── config.toml     # Streamlit 설정 (테마 등)
```

### 6-2. Streamlit Cloud 배포 순서
1. GitHub에 코드 push
2. [share.streamlit.io](https://share.streamlit.io) 접속
3. GitHub 저장소 연결
4. 배포 → 공개 URL 생성
5. 지인에게 링크 공유

## 8. 참고 사항
- Streamlit Cloud 무료 플랜: 앱 1개, 리소스 제한 있으나 개인 용도 충분

- HEIC 파일은 pillow-heif 라이브러리로 처리
- 평면도 기능은 GPS ↔ 평면도 좌표 매핑이 필요하여 별도 설계 예정
